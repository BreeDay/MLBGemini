# Gemini MLB Hackathon Submssion



## Usage

To get started, [create a free Gemini API key](https://aistudio.google.com/apikey) and add it to the `.env` file. 

REACT_APP_GEMINI_API_KEY=''

GEMINI_API_KEY=""

REACT_APP_CLIENT_ID=""

REACT_APP_SECRET_KEY=""


Then:

```
$ npm install && npm start
```
